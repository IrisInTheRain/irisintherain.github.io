细雨闲花，不及真爱如斯

——浅谈《失乐园》

  谈及最喜欢的日本文学书籍，首先闯入脑海的，是村上春树的《舞！舞！舞！》，京极夏彦的《姑获鸟之夏》和连城三纪彦的《一朵桔梗花》。一下子闯进来这么多还真是颇难抉择，毕竟每一本都爱不释手，选了一本冷落了另一本，心里总不是个滋味——干脆，哪本都不选好了——猛然发现手边正躺着渡边淳一的《失乐园》，心一横，“就它了”。



  在讲这本《失乐园》前，先提一提弥尔顿那本。作为启蒙思想先驱的弥尔顿，通过长诗《失乐园》讲述亚当和夏娃因受到撒旦的诱惑偷食禁果而被上帝逐出乐园，表达了对人类最深层的道德和信仰的批判，通过其高昂雄浑的风格传达出一种饱满的革命热情。不难想象，渡边淳一为作品取名时，也是想以此借指书中的诱惑和关于道德的思考——然而，相较于弥尔顿的积极与昂扬，渡边淳一则完全是消极的、低沉的，甚至将道德与伦理踩在脚下，表现了一种在我们的文化语境下很难理解的精神状态和价值追求。

  小说《失乐园》讲的是对妻子不再怀有感情的中年编辑久木，邂逅了受丈夫冷遇的清纯少妇凛子，二人因深入躯壳与灵魂的性爱紧密的联系在一起，炽烈地深爱着对方，最终选择在两人爱到极致的时刻一同离开这个世界。可以说，弥尔顿的“失”“乐园”是失去乐园，指离开伊甸园，而渡边淳一则讲的是让人迷失其中，不能自拔的乐园。

  把《失乐园》完全当作一本香艳小说，其中关于性爱的描写不说登峰造极，也着实称得上炉火纯青，久木的每一次爱抚，凛子的每一声呻吟，都刻画得入木三分。同时，作品又不失文学性，而且蕴含着浓厚的传统日式趣味，写春赏樱，冬赏雪，描摹风花雪月，好不美哉！换句话说，不论是世俗上，还是艺术造诣上，这本书都小有成功，而且相当卖座——但，卖座不意味着叫好。大众对这本书的看法可谓褒贬不一，溢美之词不再赘述，最为人诟病的，大概是对婚外情的讴歌和对伦理道德的漠视。

  而这本书最吸引我的，正是它展现出的离经叛道的爱情观。什么是爱情？千百年来，渴望爱情的人们、坠入爱河的人们、相依相偎的人们、白首偕老的人们无一不曾产生这样的疑问。一个人与另一个人走到一起，相知相守，究竟为的是什么？柏拉图认为是为了灵魂回忆起理型，为了精神上的水乳交融；叔本华则称爱情为“乔装打扮了的本能”，是人基于其动物性的需求复杂化而生发出的定义，显然渡边淳一站在叔本华这边，且更有升华的意味——书中描绘的性爱不再是污秽和猥琐的，而代表了至高的爱情，是纯粹而崇高的，久木和凛子因这种强烈地羁绊而密不可分，甚至超然物外，不惧生死，让人不禁认为，某种真挚的、不可名状但又堪称伟大的感情就是这种形态吧。反观柏拉图的爱情，看似美好高尚，实际上，却在不知不觉中偏了题。因为纯粹关乎精神性的满足所讲述的是爱，而非爱情——爱情之为爱情，必然包含了性的因素，否则无异于友情或亲情。用逻辑学的手段剖析开来，这三者都包含了对理型的回忆和当事人间共同创造的记忆，而爱情多出的正是性——如果说，柏拉图赞颂的是把相处的记忆也内化为理型的纯净，渡边淳一讴歌的则是把精神上的相互理解和经历的喜怒哀乐都外化为肉体间温存的纯粹。在故事的最后，久木和凛子抱在一起，在最幸福的时刻一同离开人世。这无疑把渡边想要言说的真爱表现得淋漓尽致，一则两人为了爱，可以放弃生命，这份感情的分量可见一斑，二来在最好的时光，最幸福的瞬间，欣然赴死，能这样相爱，也绝非一般人能企及。所以，无论从量，还是从质（高度），都无愧于至高的情感——真爱。

  再来看关于婚外情的控诉，我认为伦理道德的种种是人类社会发展的过程中，为了群体的和谐稳定而产生出的根深蒂固的观念，有其存在相当的理由和积极意义，但就人本身作为生物而言，则显然是非自然的，是强加给个体的枷锁，排除社会因素，人生而自由，愿一意孤行，破除世俗的桎梏本就无可厚非。如若是有自己的信仰和坚守，为了追逐心中所期待的爱情，所挂念的那个人，就更不容质疑了。人们走到一起是因为爱，婚姻在理想状况下也应该是因为爱，既然爱已不在，又何必执着。但不得不说，久木的做法着实不够磊落，扭扭捏捏，倒有些对不起凛子的一往情深，相较之下凛子则更容易被同情，因为如果说久木的所作所为最差被称为雾数，那凛子的丈夫则完全令人生厌，凛子转而爱上久木也显得相当合情。

  虽然为渡边做了这么多辩白，但仅仅建立在理解的基础上。我的爱情观与之大相径庭。我认为，无论是精神上的共鸣也好，肉体上的温存也罢，在意心灵相通的，享受鱼水之欢的，珍视过往回忆的，在乎情谊与约定的，哪怕是毫无缘由的，只要真挚热烈地与人相爱，都是一件美好的、值得祝福的事。作为社会性动物，人们总是追求相互理解，害怕孤独寂，而人生在世，人与人之间的关系总是淡薄的——人生而自私，死而自私，友情、亲情、爱情，这些情谊有时候并不那么可靠，甚至可以令人生畏，背叛、抛弃屡见不鲜。所以，能够体尝一回纯粹的爱情是多么难能可贵——不论是举案齐眉、相敬如宾，共赏春花秋月，还是缠缠绵绵、水乳交融，共度春宵良辰，亦或是书信传情，相知相惜，共话巴山夜雨，只要一个人能真正理解另一个人，哪怕方式受世俗诟病，哪怕只有须臾一瞬，都是令人艳羡的事。

用书中的话说：“真爱如此。有些人终其一生也无法遇到这样的爱。那么倘若遇到了，抛弃一切都去爱便是值得。便是最美好。”



  恬恬淡淡的感情固然美好，轰轰烈烈也未必是错。如果细雨闲花总是寂寞，还不如鼓起勇气体验一回不顾一切的爱情，哪怕结局不尽美好，也足够潇洒地说一句：“不枉此生！”
